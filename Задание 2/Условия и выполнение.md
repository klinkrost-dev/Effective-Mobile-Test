# Задание 2: Проектирование API

## Описание

Интернет-магазин «Петрушка Зеленая» развивается и открывает в мобильном приложении новый экран для отображения магазинов-партнеров (см. макеты ниже).

## Что нужно сделать

- Написать пример REST API-запроса, который будет вызываться при переходе пользователя на этот экран.
- Привести пример ответа этого REST API согласно макету (формат — JSON).
- Учесть, что при клике на карточку магазина должен осуществляться переход по ссылке на внешний ресурс.

```http

Accept: application/json
```
